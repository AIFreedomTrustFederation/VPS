# WebAI for AIFT VPS

WebAI is the assistant layer for AIFT VPS.

Its purpose is to help the project stay organized, consistent, and connected to real repository and node data.

## WebAI should help with

- README status
- documentation status
- branding status
- live data status
- node status summaries
- heartbeat summaries
- repository summaries
- template package summaries
- build status summaries
- release status summaries
- disclosure summaries

## Data sources

WebAI should use real data from:

- this repository
- connected repositories
- local registry files
- local heartbeat files
- local node cards
- local logs

## First milestone

The first WebAI milestone is a project status report that reads the repo structure and reports what exists, what is missing, and what needs attention.

## Product role

WebAI belongs inside the AIFT VPS Dashboard and the AIFT VPS Node App as the assistant that helps the owner understand the state of the network.
