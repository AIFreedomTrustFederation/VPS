## How to adapt into AIFT VPS without copying fake data

Do not copy Capital City Provisions customers, orders, leads, routes, or claims as live AIFT VPS data.

Use the repository as a real connected source for:

```text
code templates
component patterns
business workflow schemas
no-code blocks
verification commands
documentation parsing
brand rules extraction
open-source audit logic
```

Any generated AIFT VPS demo from this repo should show empty states until a real user connects real data.
