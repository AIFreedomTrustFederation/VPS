# AIFT VPS

**AIFT VPS** is the primary product brand.

AIFT VPS is a decentralized VPS network that turns phones, laptops, desktops, mini PCs, and servers into provider nodes for cloud apps, edge compute, previews, builds, and verified hosting.

## Core identity

```text
AIFT VPS = decentralized virtual private server network
```

The app foundry, dashboard, node app, templates, build system, and routing system are features of AIFT VPS.

## Product layers

```text
AIFT VPS
  -> AIFT VPS Dashboard
  -> AIFT VPS Node App
  -> AIFT VPS Provider Nodes
  -> AIFT VPS App Foundry
  -> AIFT VPS Edge Compute
```

## Technical promise

AIFT VPS gives technical users a clear model:

```text
Devices become VPS nodes.
Nodes report capabilities.
Apps and jobs run on eligible nodes.
Every workload discloses where it runs.
```

## Brand rule

Keep the term **VPS** visible in project names, docs, UI, scripts, onboarding, and README files so technical builders immediately understand the category.
